桶排序简化版

场景

班里有五个同学，满分10分，考试成绩为5,3,5,2,8，请你写个程序，将成绩按照从高到底的顺序排列。

建议你自己想想然后再看下面的。（算法了得者勿喷）

—-

—-

—-

我的思路

把所有的成绩放在数组中。for循环顺序拿出一个值命名a，然后后面的所有数字分别取出，和a相比较，大的就赋值给a，然后将赋值后的a放在新的数组中。结果是新的数组中都是8.想想明白过来，每次这样比最大的肯定都是8。每次比完的最大值不应该放到下一次的比较中。然后就陷入了怎样才能去掉这个已经比过的数值的思考中。

后来，我没想出来。。。。。。。。。。。。。

桶排序算法思路

将数组的下标作为同学的考试分数，值为有几个同学得了这个分数。
按照这样的思路来看看这些成绩5,3,5,2,8

/*
* _sortArray[0],0个同学
* _sortArray[1],0个同学
* _sortArray[2],1个同学
* _sortArray[3],1个同学
* _sortArray[4],0个同学
* _sortArray[5],2个同学
* _sortArray[6],0个同学
* _sortArray[7],0个同学
* _sortArray[8],1个同学
* _sortArray[9],0个同学
* _sortArray[10],0个同学
*/
int [] _sortArray = new int [11];

要把人数作为数组的值，下标为得分。看懂了吧。

上代码

//从小到大 
public void function1()
{
    int [] _scoreArray = new int[]{5,3,5,2,8};
    int [] _sortArray = new int [1000];
    for (int i=0;i<_scoreArray.length;i++) {
        int _a = _scoreArray[i];
        _sortArray[_a] +=1;
    }
    for (int j = 0 ;j<_sortArray.length;j++)
    {
        int _b = _sortArray[j];
        if (_b!=0)
        {
            for (int k = 0 ;k<_b;k++) {
                System.out.print(j);
            }
        }


    }
}
//从大到小
public static void function2()
{
    int [] _scoreArray = new int[]{5,3,5,2,8};
    int [] _sortArray = new int [10];
    for (int i=0;i<_scoreArray.length;i++) {
        int _a = _scoreArray[i];
        _sortArray[_a] +=1;
    }
    for (int j = _sortArray.length-1 ;j>=0;j--)
    {
        int _b = _sortArray[j];
        if (_b!=0)
        {
            for (int k = 0 ;k<_b;k++) {
                System.out.print(j);
            }
        }


    }
}

如果我们要对0-100的数进行排序我们需要101个桶。 这个是桶排序的简化版。_
