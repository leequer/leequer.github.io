冒泡排序

前情回顾

在桶排序简化版中说到桶排序。

  • 如果我们要排序5个数字，1，8，999，23，43，那我们需要建立一个长度是1000的数组。有些浪费。
  • 不能对小数进行排序。

#冒泡排序

####思路：每次比较相邻的两个数的大小，小的放后面，大的放前面，这样每轮询一次可以确定一个最小的数。

比如：8，3，4，1，6，9

首先8和3比较，小的放后面，第一次比较完毕后还是8，3，4，1，6，9。

然后接着比较3和4，小的放后面，结果是：8，4，3，1，6，9

…….

#上代码

    public static void function3() {
    int[] _scoreArray = new int[]{8, 3, 4, 1, 6, 9};
    int _size = _scoreArray.length;
    int _count = 0;
    while (_count<_size) {
        for (int i = 0; i < _size; i++) {
            int _a = _scoreArray[i];
            if (i+1>=_size)
            {
                break;
            }
            int _b = _scoreArray[i + 1];
            int _c = 0;
            if (_b > _a) {
                _c = _scoreArray[i + 1];
                _scoreArray[i + 1] = _scoreArray[i];
                _scoreArray[i] = _c;
            }
        }
        _count+=1;
    }
    for (int j =0 ; j<_size;j++)
    {
        System.out.print(_scoreArray[j]);
    }
}
