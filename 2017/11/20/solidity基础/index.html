智能合约 solidity基础

发表于 2017-11-20   |    阅读次数

引

在以太坊-私有链的安装，启动，操作一文中，我们建立了一个自己的链，运行了一些命令，简单的操作。我们知道，以太坊和比特币区别之一是可以运行智能合约。那什么是智能合约呢？

智能合约

智能合约是运行在以太坊上的一段代码。合约可以完成一些交易，查询等功能。

为什么要写智能合约

智能合约可以在满足条件的情况下，自动执行代码，并完成合约内容。不受第三方干扰。比如，保险公司一张保单，如果这保单运行在以太坊系统上，当被保人达到一定赔付要求，保单就会自动运行检查和赔付功能。不需要被保人繁琐的提交证明。而且执行过程中不会被第三方打扰，做到公平，公正。

开发语言-solidity

作为智能合约的主要官方推荐开发语言。它的编译环境Remix 在一个在线的网页上。地址是https://ethereum.github.io/browser-solidity/#optimize=false&version=soljson-v0.4.21+commit.dfe3193c.js在这里可以编辑开发智能合约。

Remix简介

reminx是solidity编写智能合约的在线ide，检查代码的正确性并执行，是在remix所在的以太坊执行这段合约。

remix概况

  • 1.代码区。编写合约。
  • 2.手动和自动编译按钮，自动编译选中后，每次改变代码，自动编译。
  • 3.当前编译的合约名字，点击Details显示合约相关信息，比如abi等。
  • 4.在1的下面，是执行后的输出区域。

run

右边切换到run的tab，

  • Environment，运行的环境，
  • Account 提供的测试用户。
  • Gas Limit 合约话费上限
  • value 如果你写的交易是需要话费gas的，那这里写gas的数量
  • 当前运行的合约

  • Create 点击此按钮时，上面Account中选择的用户则是此合约的创建用户。左边的输入框是当你的合约的构造函数中输入需要参数，那你点击create时需要输入参数。

    如图

  • 2 是此合约的地址，复制此地址填入到1的位置，点击at address，就会产生和点击create一模一样的结果。

  • 3 是点击create后产生的结果，就是你的合约中的方法，红色的是没有输出值的，就是结果不会显示在这里。灰色的就是执行完毕后会显示在这里，比如chairperson这个返回了当前合约的创建人。
